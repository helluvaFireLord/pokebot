Описание

Этот проект реализует телеграм-бота для игры в покемонов, где пользователи могут создавать, кормить и сражаться своими покемонами. Бот использует API для получения случайных покемонов, предоставляя изображения и имена, а также уникальные характеристики для каждого игрока.

Основные функции бота:

 1. Создание покемона: Игрок может создать покемона случайного типа: обычный покемон, волшебник или боец.
 2. Просмотр информации: Команда для отображения информации о созданном покемоне.
 3. Кормление покемона: Кормление восстанавливает здоровье покемона.
 4. Атака: Игрок может атаковать других покемонов, если ответит на сообщение другого игрока.

Основные команды:

 • /start - Стартовое сообщение, приветствует пользователя и предлагает создать покемона.
 • /go - Создание покемона (Pokemon, Wizard или Fighter).
 ![Telegram_13Wj3n1QnA](https://github.com/user-attachments/assets/2f00fd16-be09-4ef1-b9df-fbce970fb2da)

 • /info - Вывод информации о покемоне (имя, сила, здоровье).
 ![Telegram_cB6GbYSeFR](https://github.com/user-attachments/assets/2e5cdf90-fac7-419c-9cfc-db3dd56954a5)

 • /feed - Кормление покемона для восстановления его здоровья.
![Telegram_Bt95ja9VnX](https://github.com/user-attachments/assets/386dcbd3-74c7-452d-a536-c6ee7ba925d0)

 
 • /atk - Атака другого покемона (нужно ответить на сообщение игрока, чтобы атаковать его покемона).

Пример использования

 1. Игрок запускает бота командой /start.
 2. Создает себе покемона командой /go.
 3. Просматривает информацию о своём покемоне с помощью команды /info.
 4. Кормит своего покемона, восстанавливая здоровье через /feed.
 5. Атакует другого игрока, используя команду /atk, отвечая на сообщение другого игрока.

Уникальные особенности покемонов:

 • Обычный покемон - стандартная атака и восстановление здоровья.
 • Wizard - шанс применить щит в бою.
 • Fighter - может использовать супер-атаку с повышенной силой.
